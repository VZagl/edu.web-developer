}
```

–ú–æ–∂–Ω–æ —É–∫–∞–∑–∞—Ç—å –¥–æ–ø—É—Å—Ç–∏–º—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è –¥–ª—è —à–æ—Ä—Ç–∫–∞—Ç–∞ `grid-template`:

–í —Ç–æ–º —á–∏—Å–ª–µ –º–æ–∂–Ω–æ —É–∫–∞–∑–∞—Ç—å –∏–º–µ–Ω–∞ –ª–∏–Ω–∏–π:

–ú–æ–∂–Ω–æ –∑–∞–¥–∞—Ç—å —Ä–∞–∑–º–µ—Ä—ã –∫–æ–ª–æ–Ω–æ–∫ –∏ —Ä—è–¥–æ–≤. –°–æ–∑–¥–∞–¥–∏–º –¥–≤–∞ —Ä—è–¥–∞ –∏ –¥–≤–µ –∫–æ–ª–æ–Ω–∫–∏:

`auto-flow` ‚Äî —ç—Ç–æ –∫–ª—é—á–µ–≤–æ–µ —Å–ª–æ–≤–æ –¥–∞—ë—Ç –±—Ä–∞—É–∑–µ—Ä—É –ø–æ–Ω—è—Ç—å, —á—Ç–æ —Å–æ–∑–¥–∞–≤–∞—Ç—å –ø—Ä–∏ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç–∏: –∫–æ–ª–æ–Ω–∫–∏ –∏–ª–∏ —Ä—è–¥—ã.

	display: grid;
	grid: 200px 100px / auto-flow 30%;
}
```

–ï—Å–ª–∏ `auto-flow` —Å—Ç–æ–∏—Ç —Å–ª–µ–≤–∞ –æ—Ç —Å–ª—ç—à–∞, —Ç–æ –±—É–¥—É—Ç —Å–æ–∑–¥–∞–≤–∞—Ç—å—Å—è –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏–µ —Ä—è–¥—ã:

`dense` ‚Äî –∫ –∫–ª—é—á–µ–≤–æ–º—É —Å–ª–æ–≤—É `auto-flow` –º–æ–∂–Ω–æ –¥–æ–±–∞–≤–∏—Ç—å `dense`. –≠—Ç–æ —É–∫–∞–∂–µ—Ç –±—Ä–∞—É–∑–µ—Ä—É, —á—Ç–æ —ç–ª–µ–º–µ–Ω—Ç—ã –¥–æ–ª–∂–Ω—ã —Å—Ç–∞—Ä–∞—Ç—å—Å—è –∑–∞–ø–æ–ª–Ω–∏—Ç—å —Å–≤–æ–±–æ–¥–Ω—ã–µ —è—á–µ–π–∫–∏. –ü–æ–¥—Ä–æ–±–Ω–µ–µ –ø—Ä–æ —Ä–∞–±–æ—Ç—É —ç—Ç–æ–≥–æ –∫–ª—é—á–µ–≤–æ–≥–æ —Å–ª–æ–≤–∞ –º–æ–∂–Ω–æ –ø–æ—á–∏—Ç–∞—Ç—å –≤ —Å—Ç–∞—Ç—å–µ –ø—Ä–æ [`grid-auto-flow`](https://doka.guide/css/grid-auto-flow/).

–í–æ–∑–º–æ–∂–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è:

- `auto` (–∑–Ω–∞—á–µ–Ω–∏–µ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é) ‚Äî —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ, –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –¥–ª—è —Å–±—Ä–æ—Å–∞ —Ä–∞–Ω–µ–µ –∑–∞–¥–∞–Ω–Ω—ã—Ö –∑–Ω–∞—á–µ–Ω–∏–π.
- `align-self justify-self` ‚Äî –ø–µ—Ä–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –∑–∞–¥–∞—ë—Ç –∑–Ω–∞—á–µ–Ω–∏–µ —Å–≤–æ–π—Å—Ç–≤—É `align-self`, –≤—Ç–æ—Ä–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ —É—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ —Å–≤–æ–π—Å—Ç–≤–∞ `justify-self`. –ï—Å–ª–∏ —É–∫–∞–∑–∞–Ω–æ –≤—Å–µ–≥–æ –æ–¥–Ω–æ –∑–Ω–∞—á–µ–Ω–∏–µ, —Ç–æ –æ–Ω–æ —É—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ—Ç—Å—è –¥–ª—è –æ–±–æ–∏—Ö —Å–≤–æ–π—Å—Ç–≤. –ù–∞–ø—Ä–∏–º–µ—Ä, `place-self: center` –æ—Ç—Ü–µ–Ω—Ç—Ä–∏—Ä—É–µ—Ç —ç–ª–µ–º–µ–Ω—Ç –ø–æ –≥–æ—Ä–∏–∑–æ–Ω—Ç–∞–ª—å–Ω–æ–π –∏ –ø–æ –≤–µ—Ä—Ç–∏–∫–∞–ª—å–Ω–æ–π –æ—Å—è–º –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ.

### [–°–ø–µ—Ü–∏–∞–ª—å–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏ –∏ –∫–ª—é—á–µ–≤—ã–µ —Å–ª–æ–≤–∞](https://doka.guide/css/grid-guide/#specialnye-funkcii-i-klyuchevye-slova)

–ö–æ–≥–¥–∞ –≤—ã –∑–∞–¥–∞—ë—Ç–µ —Ä–∞–∑–º–µ—Ä—ã –∫–æ–ª–æ–Ω–∫–∞–º –∏ —Ä—è–¥–∞–º, –≤–∞–º –¥–æ—Å—Ç—É–ø–Ω—ã –Ω–µ —Ç–æ–ª—å–∫–æ –¥–∞–≤–Ω–æ –∏–∑–≤–µ—Å—Ç–Ω—ã–µ –µ–¥–∏–Ω–∏—Ü—ã –∏–∑–º–µ—Ä–µ–Ω–∏—è (`px`, `vw`, `rem`, `%` –∏ —Ç–∞–∫ –¥–∞–ª–µ–µ), –Ω–æ –∏ –æ—á–µ–Ω—å –∫—Ä—É—Ç—ã–µ –∫–ª—é—á–µ–≤—ã–µ —Å–ª–æ–≤–∞ `min-content`, `max-content` –∏ `auto`. –ò —É–∂–µ —É–ø–æ–º—è–Ω—É—Ç—ã–µ –µ–¥–∏–Ω–∏—Ü—ã –∏–∑–º–µ—Ä–µ–Ω–∏—è `fr`.

–ì—Ä–∏–¥—ã –ø–æ–¥–∞—Ä–∏–ª–∏ –Ω–∞–º –µ—â—ë –æ–¥–Ω—É –≤–æ–ª—à–µ–±–Ω—É—é —Ñ—É–Ω–∫—Ü–∏—é, –ø–æ–∑–≤–æ–ª—è—é—â—É—é –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ –∑–∞–¥–∞–≤–∞—Ç—å –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–π –∏ –º–∞–∫—Å–∏–º–∞–ª—å–Ω—ã–π —Ä–∞–∑–º–µ—Ä ‚Äî `minmax()`. –ù–∞–ø—Ä–∏–º–µ—Ä, –≤ —Å–ª—É—á–∞–µ –∑–∞–ø–∏—Å–∏ `grid-template-columns: minmax(200px, 1fr);` –∫–æ–ª–æ–Ω–∫–∞ –∑–∞–π–º—ë—Ç 1 —á–∞—Å—Ç—å —Å–≤–æ–±–æ–¥–Ω–æ–≥–æ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–∞ –≥—Ä–∏–¥-–∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–∞, –Ω–æ –Ω–µ –º–µ–Ω—å—à–µ 200 –ø–∏–∫—Å–µ–ª–µ–π.

–ï—â—ë –æ–¥–Ω–∞ –ø–æ–ª–µ–∑–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è, –ø–æ—è–≤–∏–≤—à–∞—è—Å—è –≤ –≥—Ä–∏–¥–∞—Ö, —ç—Ç–æ `repeat()`. –°—ç–∫–æ–Ω–æ–º–∏—Ç –≤–∞–º –∫—É—á—É –ª–∏—à–Ω–∏—Ö –±—É–∫–≤ –∏ –≤—Ä–µ–º–µ–Ω–∏.

### [–ê–Ω–∏–º–∞—Ü–∏—è —Å–≤–æ–π—Å—Ç–≤](https://doka.guide/css/grid-guide/#animaciya-svoystv)

–î–ª—è –∞–Ω–∏–º–∞—Ü–∏–∏ –¥–æ—Å—Ç—É–ø–Ω—ã —Å–ª–µ–¥—É—é—â–∏–µ —Å–≤–æ–π—Å—Ç–≤–∞ –∏ –∏—Ö –∑–Ω–∞—á–µ–Ω–∏—è:

- –ó–Ω–∞—á–µ–Ω–∏—è —Å–≤–æ–π—Å—Ç–≤ `gap`, `row-gap`, `column-gap`, —É–∫–∞–∑–∞–Ω–Ω—ã–µ –≤ –µ–¥–∏–Ω–∏—Ü–∞—Ö –∏–∑–º–µ—Ä–µ–Ω–∏—è, –ø—Ä–æ—Ü–µ–Ω—Ç–∞—Ö –∏–ª–∏ –ø—Ä–∏ –ø–æ–º–æ—â–∏ `calc()`.
- –ó–Ω–∞—á–µ–Ω–∏—è —Å–≤–æ–π—Å—Ç–≤ `grid-template-columns`, `grid-template-rows`, —É–∫–∞–∑–∞–Ω–Ω—ã–µ –≤ –µ–¥–∏–Ω–∏—Ü–∞—Ö –∏–∑–º–µ—Ä–µ–Ω–∏—è, –ø—Ä–æ—Ü–µ–Ω—Ç–∞—Ö –∏–ª–∏ –ø—Ä–∏ –ø–æ–º–æ—â–∏ —Ñ—É–Ω–∫—Ü–∏–∏ `calc()` –ø—Ä–∏ —É—Å–ª–æ–≤–∏–∏, —á—Ç–æ –∞–Ω–∏–º–∏—Ä—É—é—Ç—Å—è –∞–Ω–∞–ª–æ–≥–∏—á–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è.

### [–ù–∞ –ø—Ä–∞–∫—Ç–∏–∫–µ](https://doka.guide/css/grid-guide/#na-praktike)
